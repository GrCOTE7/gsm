## Raccourcis usuels

Liste non exhaustive :

```dos
ALT + 3 → 31 : ♥ ... ▼
ALT + 24 à 27: ↑ ↓ → ←
ALT + 144, 183 : É À
MAJ + ALT + ↑ ou ↓ : COPIÉ/COLLÉ décalé d'une ligne
ALT + ← : Retourner au précédent code édité (historique)
ALT + → : Revenir au dernier code édité

CTRL + ALT + S : Surround
CTRL + u + u : Min/MAJ switch (bascule)
```
