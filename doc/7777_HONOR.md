### LV - **L**e**V**el - Niveau

Pour ajouter votre propre progression :

- Positionner le curseur sur la ligne
du dernier contributeur,
- fair MAJ + ALT + ↓,
- Modifier les caractères pour y laisser votre pseudo + Niveau atteint
  ('Le numéro du DOC dans lequel vous êtes - Ex. 0305 = GIT Approfondi, palier 5,
  ce qui suppose l'intégralité des précédents niveaux maîtrisés)
